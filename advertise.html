合作資訊｜刊登說明

合作資訊欄

Tenggo Tools 提供自製工具內的合作資訊欄位。內容會依實際工具畫面配置安排， 目前提供工具視窗底部橫幅資訊欄，適合工具、素材服務、美術委託、網站服務、技術交流社群、教學內容或相關活動資訊曝光。

刊登總覽
單一工具最多 3 張輪播
每張約 10 秒切換
支援自備圖 / AI 輔助代製

可刊登版位

  • Tenggo Tools 旗下自製工具可依工具介面配置合作資訊欄位。
  • 目前主要提供：工具程式視窗底部橫幅資訊欄。
  • 顯示位置：工具程式視窗底部，不影響主要操作功能。
  • 顯示形式：僅支援靜態圖片輪播，不支援影片、GIF、動畫或互動式廣告。
  • 單一工具內最多輪播 3 張圖片，每張圖片可對應不同合作資訊。
  • 輪播時間：每張圖片顯示約 10 秒後切換下一張。
  • 可選擇單一輪播版位刊登，或洽談 3 個輪播版位全包合作。
  • 每張圖片皆可設定點擊後開啟指定網址。
  • 其他工具完成並測試穩定後，將陸續開放合作資訊欄位。

聯絡方式

  • 合作資訊詢問：eric.0955@gmail.com
  • 來信標題建議：[Tenggo Tools 合作資訊] 合作方名稱
  • 若只是先詢問版位、費用或刊登方式，也可以先提供簡單需求。
  • 正式刊登前，會再確認圖片、連結、刊登期間、版位數量與付款方式。

圖片規格建議

  • 單一工具內底部橫幅：建議 1200 × 72 px。
  • 圖片格式：PNG / JPG，請提供靜態圖片檔。
  • 輪播數量：單一工具內最多 3 張圖片。
  • 輪播時間：每張圖片約 10 秒。
  • 內容建議：名稱、服務主題、簡短標語或活動資訊。
  • 文字不宜過多，避免縮小後難以閱讀。
  • 若有點擊連結，請提供每張圖片對應的完整網址。

參考費用

  • 單一工具內，單一輪播版位：NT$800 起 / 月。
  • 單一工具內，單一輪播版位短期測試：NT$300 起 / 週。
  • 單一工具內，3 個輪播版位全包：NT$2,000 起 / 月。
  • 單一工具內，3 個輪播版位短期全包：NT$800 起 / 週。
  • 以上為初期參考費用，實際費用會依工具類型、刊登期間、曝光位置、輪播數量與素材需求調整。
  • 長期合作、連續刊登或多工具刊登，可另行討論合作方案。
  • 正式刊登前會先確認圖片、連結、刊登期間、版位數量與是否需要代製圖片。

製圖方式

  • 自備圖片：由合作方自行提供符合規格的靜態圖片。
  • 代製圖片：可委託 Tenggo Tools 以 AI 輔助設計方式製作簡易靜態橫幅圖片，並進行文字排版與尺寸調整。
  • 目前僅支援 PNG / JPG 靜態圖片，不支援影片、GIF、動畫或互動式廣告。
  • 若圖片需使用指定 Logo、商標、人物、產品圖、遊戲素材或品牌素材，請由合作方自行提供，並確認具備合法使用權或授權。
  • 建議尺寸：單一工具內底部橫幅 1200 × 72 px。
  • 圖片格式：PNG / JPG，請提供靜態圖片檔。
  • 若為輪播刊登，請依刊登版位數量提供對應圖片。

製圖費用

  • 自備圖片：不另收製圖費。
  • 簡易靜態圖片代製：NT$300 起 / 張，採 AI 輔助設計與人工排版調整。
  • 同風格延伸製作：NT$200 起 / 張。
  • 若需多次修改、特殊設計或大量文字排版，費用另議。
  • 正式製作前會先確認文案、圖片素材、品牌色與刊登用途。

付款方式

  • 合作刊登費用需於刊登前全額支付，確認付款後才會安排刊登。
  • 若需代製圖片，製圖費用需於開始製作前支付。
  • 簡易靜態圖片代製包含 1 次小幅修改；若需多次修改、重新設計或更換方向，費用另議。
  • 付款方式：銀行轉帳 / LINE Pay，確認合作後提供付款資訊。
  • 付款完成後，請提供付款資訊或截圖以利確認。

付款與刊登確認

  • 正式刊登前會先確認圖片、連結、刊登期間與版位數量。
  • 若為 3 個輪播版位全包，需於刊登前確認 3 張圖片與對應連結。
  • 刊登開始後，若需臨時更換圖片或連結，請提前聯絡確認。
  • 若內容不符合刊登規範，Tenggo Tools 保留拒絕刊登、要求修改或提前下架之權利。

可接受的合作內容

  • 工具、軟體、素材整理、圖檔轉換或技術教學內容。
  • 美術委託、圖示製作、介面設計、網站架設等服務。
  • 技術交流社群、教學頻道、玩家交流社群或非商業活動資訊。
  • 其他與工具使用者相關，且內容合法、清楚、不誤導使用者的合作資訊。

不接受的刊登內容

  • 博弈、賭博、詐騙、成人內容、惡意軟體、釣魚網站或高風險連結。
  • 外掛、破解、盜版、未授權程式、未授權客戶端或登入器下載。
  • 元寶、裝備、帳號、代儲、現金交易或其他可能涉及爭議的交易內容。
  • 未經授權使用他人商標、Logo、角色圖、遊戲截圖、音樂、圖片或其他受保護素材。
  • 宣稱官方授權、保證獲利、保證安全、保證不封鎖等無法提出證明的內容。

合作前請提供

  • 合作方名稱或品牌名稱。
  • 希望刊登的工具名稱，例如：L1 Image Converter。
  • 預計刊登期間，例如：1 週 / 1 個月。
  • 刊登方式：單一輪播版位或 3 個輪播版位全包。
  • 製圖方式：自備圖片或委託代製圖。
  • 若需使用指定 Logo、商標、人物、產品圖、遊戲素材或品牌素材，請一併提供可合法使用的素材檔案。
  • 刊登用靜態圖片檔案，格式為 PNG / JPG。
  • 每張圖片點擊後要開啟的網址。
  • 聯絡方式。

刊登流程

  • 1. 聯絡並提供合作需求。
  • 2. 確認刊登工具、版位數量、期間與費用。
  • 3. 提供圖片與連結網址。
  • 4. 完成內容審核與版面確認。
  • 5. 於指定版本或指定期間內刊登。

刊登提醒

  • 圖片不得偽裝成下載按鈕。
  • 不得放置惡意軟體或釣魚連結。
  • 不得誤導使用者下載非本站工具。
  • 不得使用容易造成使用者誤點的設計。
  • 合作方需自行確認提供的文字、圖片、連結與服務內容具備合法權利。
  • Tenggo Tools 保留審核、拒絕刊登、要求修改與提前下架內容之權利。
